Adding an event

To add an event just replace the day li tag with the following. Or you can add the missing elements (highlighted in red). 

  <li class="day day--withEvent">
    <div class="dayForMobile">Monday, February</div>
      <div class="dayDate">4</div>
      <div class="event">
          <h5>Indians <div class="homeTeamForMobile">@ LA</div></h5>
          <a href="#" class="button tiny">Tickets</a>
          <p class="eventTime">1:07</p>
        </div>
  </li>

Explanation of calendar elements

<ul class="accordion" data-accordion data-multi-expand="true" data-allow-all-closed="true">

The accordion parent container that houses all of the collapseable elements. (The data attributes define whether it allows multiple accordion elements open at once, and whether it allows months to be closed at one time.)

<li class="accordion-item bpCalendar is-active" data-accordion-item><!-- BEGIN FEB -->

The accordion container for all the month accordion content.

<a name="february"></a>

A tag used for anchor links to jump to this specific month within the page.

<a href="#" class="bpCalendar__titleHeading">February 2019</a>

Month title and clickable element that triggers open/close of accordion.

<div class="accordion-content" data-calendar-hide-date="2019-3-1" data-tab-content>

Content of accordion that is hidden and shown based on current status. The data-calendar-hide-date defines when the calendar will be automatically collapsed on page load.
