DataTables

To add the basic Data Tables sorting functionality just add the dataTable class to your table. The table html syntax must be correct for the plugin to work properly.

To add the right side shadow to indicate additional content on smaller screens, wrap the table in a div with the class scrolling-table. Can be used with or without the DataTables plugin.

Visibility Classes

The classes below can be used to hide/show content based on the width of the user's browser. They can be applied to any element.

There are 3 defined sizes: small (0 - 640px), medium (640px - 1024px), large (1024px - 1200px), and xlarge (1200px and up). This is a SMALL MEDIUM LARGE XLARGE screen.
